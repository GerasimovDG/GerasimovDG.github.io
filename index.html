Movie Ticket System
Язык программирования: JavaScript
Описание: Работа написана с помощью фреймворка Angular. Имитирует сервис по продаже билетов в кино. Есть фильтр по интересующим вас позициям: дате, рейтингу, кинотеатру, жанру, времени. Можно выбрать кинотеатр, в котором показывают данный фильм, сеанс и место в зале. Также имитируется покупка билетов. Есть проверка форм при авторизации/регистрации. При выборе мест и покупке билета также есть валидация форм и различные проверки.
Данные для входа: Логин - admin, Пароль - admin1
